- [ ] LinkedIn post text finalized — hook reviewed (does it make you stop scrolling?)

- [ ] Video recorded and edited (captions on, intro/outro card added)

- [ ] LinkedIn post published at 07:30 GMT

AMPLIFY monitors and responds. Every comment gets a reply. Every "can you share the code?" gets the GitHub link. Every "I'd love to see this applied to X" gets "DM me — I run workshops on exactly this."
